$ man me — the 10 commands I reach for daily

Top 10 Terminal Commands

The ten commands I reach for most — each paired with a one-line description and a real moment you'd use it. Tap any command to copy it.

  1. 01
    $ cd

    Move between folders — your main way of navigating the filesystem.

    whenjump straight into a project: cd ~/projects/tradeacademy

  2. 02
    $ ls -la

    List everything in a folder — hidden files and permissions included.

    whenchecking whether a .env exists and who can read it: ls -la

  3. 03
    $ grep -r

    Search text or patterns across files, recursively into subfolders.

    whenfinding every loose end in the code: grep -r "TODO" src/

  4. 04
    $ find

    Locate files by name, type, or age — anywhere under a directory.

    whenhunting stray logs before a commit: find . -name "*.log"

  5. 05
    $ tail -f

    Stream the end of a file live as new lines are written.

    whenwatching Spring Boot logs while you test an endpoint: tail -f app.log

  6. 06
    $ curl

    Make HTTP requests straight from the terminal — no Postman needed.

    whenpinging your running API: curl localhost:8080/api/health

  7. 07
    $ ps aux | grep

    List running processes and filter down to the one you care about.

    whenfinding a stuck Java process and its PID: ps aux | grep java

  8. 08
    $ kill

    Stop a process by its PID when it won't quit on its own.

    whenfreeing a hung server holding port 8080: kill -9 12345

  9. 09
    $ chmod +x

    Change file permissions — most often, make a script executable.

    whenso you can run it directly: chmod +x deploy.sh → ./deploy.sh

  10. 10
    $ ssh

    Securely log into a remote machine from your own terminal.

    whenconnecting to your deployment box: ssh prajwal@server-ip
